# 🐧 Linux Shell Scripting Programs

## Overview

This repository contains a collection of basic Linux Shell Scripts developed to practice Bash scripting concepts and fundamental programming logic. The scripts demonstrate the use of variables, loops, conditional statements, string operations, arithmetic calculations, and file/directory handling in Linux.

These programs are suitable for beginners learning Linux command-line programming and shell scripting.

---

## Technologies Used

* Linux
* Bash Shell Scripting
* GNU/Linux Command Line Utilities

---

## Programs Included

| Script                | Description                                    |
| --------------------- | ---------------------------------------------- |
| `factorial.sh`        | Calculates the factorial of a given number     |
| `fabonacci.sh`        | Generates the Fibonacci sequence               |
| `armstrong.sh`        | Checks whether a number is an Armstrong number |
| `palindrome.sh`       | Checks whether a number is a palindrome        |
| `stringpalindrome.sh` | Checks whether a string is a palindrome        |
| `primenumber.sh`      | Determines whether a number is prime           |
| `reverse string.sh`   | Reverses a given string                        |
| `string concatinate`  | Concatenates two strings                       |
| `operation.sh`        | Performs basic arithmetic operations           |
| `loop.sh`             | Demonstrates looping constructs in Bash        |
| `directory.sh`        | Performs basic directory operations            |

## How to Run

### Clone the Repository

```bash
git clone https://github.com/Khushi966/linux.git
cd linux
```

### Give Execute Permission

```bash
chmod +x filename.sh
```

### Execute the Script

```bash
./filename.sh
```

or

```bash
bash filename.sh
```
